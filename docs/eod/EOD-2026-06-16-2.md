### Bug fixes

- **Epoch-milliseconds in SQLite TEXT columns**: Hibernate's `TimestampJdbcType`
  was writing `LocalDateTime` and `LocalDate` values as numeric epoch-ms strings
  into SQLite TEXT columns, causing parse failures on the next read. Created
  `LocalDateTimeConverter` and `LocalDateConverter` with `@Converter(autoApply=true)`;
  both include an epoch-ms fallback so existing data (if any) round-trips safely.
- **Nav highlight showing Dashboard active on detail screen**: `showView()` (used
  to push sub-screens like MemberDetail) was not moving keyboard focus away from
  the Dashboard nav button. Fixed by calling `contentArea.requestFocus()` in both
  `navigateTo()` and `showView()`, and by tracking the active CSS class explicitly
  in `setActiveNav()` — the Members button now stays highlighted while the detail
  form is open.
- **Double-submit on Save**: added `saveButton.setDisable(true)` at the top of
  `onSave()`; re-enabled only on error.
